# Frontend — Sistema de Compras Online

## Instalação

1. Instale as dependências do frontend:

```bash
npm install
```

2. Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

3. Acesse o frontend em:

```bash
http://localhost:5173
```

## Notas

- O frontend utiliza React, TypeScript e Tailwind CSS.
- A API deve estar rodando em `http://localhost:8000` para buscar produtos.
